# 万物皆可绳 / Super Lead

[English](README.md)

**一根绳子，所有连接。**

万物皆可绳是一个 NeoForge 模组，把原版拴绳变成万能连接工具。红石信号、能量、物品、流体——全靠一根绳子。

---

## 绳子种类

用对应材料右键绳子即可原地转换或升级。

| 类型 | 转换材料 | 升级材料 | 效果 |
|------|---------|---------|------|
| **红石** | 红石块 | — | 端点间双向红石信号 |
| **能量 (FE)** | 铁块 | 红石块 | FE 传输，tier 翻倍吞吐 (1x→2x→4x…) |
| **物品** | 漏斗 | 箱子 | 物品运输，栅栏结多绳轮询路由 |
| **流体** | 炼药锅 | 桶 | 容器间流体传输 |
| **压力 (气体)** | 钢块 | 强化合金 | Mekanism 化学/气体传输 |
| **热量** | 铜块 | 强化合金 | Mekanism 热量传输 |
| **AE 网络** | 福鲁伊克斯块 | 16³ 空间组件 | AE2 频道桥接 |

### 能量绳
- 基础吞吐：tier 0 = 256 FE/t，每级翻倍
- 最大 tier 可配置（默认 30）
- 通电绳子发光并喷发粒子

### 物品/流体绳
- 用转换材料点击端点切换提取方向
- tier 0: 1 个物品 / 1000 mB 每次；每级翻倍
- 栅栏结多绳连接自动轮询分流

## 用法

### 放置绳子
右键方块面 → 设置第一锚点。右键另一方块 → 创建连接。锚点可贴在任意方块面，不限于栅栏。

### 延伸长度
新绳子：点击方块设置第一锚点后，**再次点击同一锚点**可消耗 1 根拴绳延长 1 倍距离（最多 4 倍）。已有绳子：潜行 + 右键锚点延伸。延伸后的绳子升级消耗翻倍。

### 剪断返还
剪断绳子时按延伸倍率返还对应数量的拴绳（2 倍长返还 2 根，4 倍长返还 4 根）。

### 悬挂物品
主手持可悬挂物品（灯笼、告示牌、方块），副手持线。右键绳子即可悬挂。告示牌可在世界中编辑。可切换方块/物品显示模式。

### 剪断
剪刀右键绳子 → 剪断并掉落。剪刀右键栅栏结 → 剪断结上所有绳子。

### 滑索
手持锁链右键绳子即可开始滑索骑行。

## 物理

客户端 Verlet 积分模拟：重力、阻尼、方块碰撞、绳子间互斥。静止后自动冻结——闲置时零 CPU 开销。

## 配置

`config/super_lead-common.toml`：

| 设置 | 默认 | 说明 |
|------|------|------|
| `energy.base_transfer_per_tick` | 256 | tier 0 时基础 FE/t |
| `energy.tier_max_level` | 30 | 能量绳最大升级 tier |
| `network.max_leash_distance` | 12.0 | 每长度单位格数 |
| `network.max_ropes_per_block_face` | 8 | 每个方块面最多绳子数 |
| `network.item_transfer_interval_ticks` | 4 | 物品传输间隔 tick |
| `network.fluid_bucket_amount` | 1000 | 每次流体传输 mB |
