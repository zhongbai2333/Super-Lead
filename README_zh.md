# 万物皆可绳 / Super Lead

[English](README.md)

**万物皆可绳** 是一个围绕 Minecraft 原版拴绳展开的 NeoForge 模组。

你有没有想过，拴绳除了拴猪，还能干点别的？

这个模组优化了原版拴绳——红石传导、能量输送、物品运输、流体流通，全都靠一根绳子搞定。

操作简单，视觉直观，还带着那条原版的小垂线。

毕竟，连接本来就不需要那么复杂。

## 核心理念

Super Lead 保留了原版拴绳最直观的体验：两个点，一条看得见的连接线。它不是再做一套占方块空间的管道系统，也不是把连接关系藏起来的无线传输，而是把拴绳变成一种更轻量、更直观的工程语言。

设计目标是让它看起来像是 Minecraft 里自然长出来的机制：

- 仍然使用原版拴绳作为基础物品；
- 允许拴绳固定到方块任意面；
- 保留原版拴绳的下垂曲线；
- 连接关系直接显示在世界中；
- 基础操作不依赖复杂 UI；
- 通过距离、容量和单功能升级控制强度。

## 已实现功能

### 方块到方块连接

手持拴绳右键一个方块面，选择第一端锚点；再右键另一个方块，即可建立两点之间的连接。

锚点会根据方块形状和点击面自动计算，让普通方块、非完整方块、机器方块、栅栏和装饰方块上的绳子落点尽量自然。

连接数据会随世界保存，并同步给客户端。

### 物理绳子渲染

模组使用自定义绳子几何，不再完全复用原版扁平 leash 条带。客户端使用 Verlet 风格的绳子模拟，包含重力、阻尼、距离约束、方块碰撞和绳子之间的分离。

最终效果是一条可视、轻量，并且仍然具有原版拴绳气质的连接线。

### 红石拴绳

第一套功能拴绳是 **红石拴绳**。

- 使用红石块升级原版拴绳。
- 用红石块右键已有普通连接，可以把这条连接升级为红石连接。
- 红石拴绳显示为红色与深红色渐变。
- 通电后颜色会变亮，并在视觉上发光。
- 通电时会随机产生少量红石粒子。
- 红石信号可以在连接两端之间传输。

当玩家手持红石块瞄准可升级的普通绳子时，目标连接会显示红色虚影。手持剪刀瞄准连接时，则会显示剪断预览虚影。

### 剪断连接

使用剪刀右键世界中的绳子连接即可剪断，并掉落对应的拴绳。

剪断和红石升级现在共用同一套“连接动作”逻辑，后续添加新的材料升级或工具交互会更简单。

## 规划中的功能拴绳

本模组采用“同一基础物品，不同功能升级”的设计。每根绳子同一时间只拥有一种功能，方便玩家理解，也方便通过颜色快速识别用途。

| 类型 | 示例材料 | 视觉方向 | 用途 |
| --- | --- | --- | --- |
| 红石拴绳 | 红石块 | 红色 / 深红色 | 传递红石信号 |
| 物品拴绳 | 漏斗 / 铁锭 | 灰色 | 在容器之间低速传输物品 |
| 能量拴绳 | 铜锭 / 红石材料 | 橙色 | 传输 FE 或兼容能量 |
| 流体拴绳 | 水桶 / 玻璃瓶 | 蓝色 | 连接流体容器 |

## 平衡设计

万物皆可绳并不打算直接替代所有管道、线缆或物流模组。它更强调灵活、可视和原版感。

当前与后续的平衡方向包括：

- 限制最大连接距离；
- 每根绳子只拥有一种功能；
- 连接必须在世界中可见；
- 吞吐量低于专业物流系统；
- 两端必须具备对应能力；
- 无效、过远或方块被破坏的连接会被清理。

## 开发路线

### 里程碑 1：可连接原型

- 支持方块面锚点。
- 连接可保存和加载。
- 世界中能看到绳子。

### 里程碑 2：功能拴绳

- 加入红石拴绳升级。
- 加入功能颜色识别。
- 加入升级与剪断预览。

### 里程碑 3：传输系统

- 实现物品、能量或流体传输。
- 加入距离、容量和速度配置。
- 完善方块破坏、区块卸载、维度切换等边界情况。

### 里程碑 4：展示打磨

- 补充粒子、音效和提示文本。
- 制作展示场景和 README 截图。
- 考虑加入中继节点，支持更长的链式网络。

## 为什么不是管道？

Super Lead 的重点不是做又一个管道系统，而是把 Minecraft 里最普通、最有“连接感”的物品改造成工程工具。

- 比管道更轻量，不占完整方块空间。
- 比无线传输更直观，连接关系直接出现在世界里。
- 比全新系统更原版，玩家一拿到拴绳就知道该怎么试。

一根普通的绳子，连接不普通的世界。

## 环境需求

- Minecraft：`26.1.2`
- NeoForge：`26.1.2.43-beta` 或兼容的 `26.x`
- Java：`25`

## 构建

这是一个基于 Gradle 的 NeoForge 项目。需要本地编译时，在仓库根目录运行 Gradle 构建任务即可。
