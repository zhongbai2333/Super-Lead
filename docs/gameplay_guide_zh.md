# 《万物皆可绳》游玩指南

> 一根绳子，连接方块、机器、网络，也顺便连接一点离谱的工程学。

本指南适用于当前仓库版本 **Super Lead 0.3.1-beta**，以实际实现为准。默认数值可能被服务器配置修改。

## 运行环境

- Minecraft 26.1.2
- NeoForge 26.1.2.76（模组元数据兼容 NeoForge 26.x）
- Java 25
- 客户端和服务端均需安装

以下模组为可选联动：

- Applied Energistics 2：ME 线缆拴绳、绳挂 ME 终端
- Mekanism：加压管道拴绳、热导管道拴绳及部分流体机器兼容
- JEI：向货单拖入物品样本
- NetMusic / NetMusicCanPlayBili：音乐设备自动装入与抽取

## 五分钟快速上手

### 1. 制作超级拴绳

将 **1 根原版拴绳**放入合成栏，可得到 **1 根超级拴绳**。这是所有功能绳的基础物品。

### 2. 连接两个方块面

1. 手持超级拴绳，右键第一个方块面，设置第一锚点。
2. 右键另一个方块面，完成连接。
3. 生存模式下，连接成功后消耗 1 根超级拴绳。

锚点可以位于普通方块、容器、机器或栅栏等位置。连接机器时，选择的方块面很重要：不少机器只会在特定面开放输入或输出。

默认情况下，每个长度单位允许连接 **12 格**。

### 3. 放置时延长绳索

选择第一锚点后，再次点击同一锚点，可以额外投入 1 根同类型超级拴绳并增加一个长度单位，最多增加到 4 个长度单位。

默认最大距离为：

| 长度单位 | 最大距离 | 总计投入的绳 |
| ---: | ---: | ---: |
| 1 | 12 格 | 1 根 |
| 2 | 24 格 | 2 根 |
| 3 | 36 格 | 3 根 |
| 4 | 48 格 | 4 根 |

取消放置、超出距离、切换维度或退出时，放置阶段额外投入的绳会返还。

### 4. 把普通绳变成功能绳

放好绳后，**潜行并手持对应材料右键绳体**即可转换。准星指向绳索时会显示类型、等级和可执行操作。

| 绳索 | 转换材料 | 主要用途 |
| --- | --- | --- |
| 红石拴绳 | 红石块 | 双向传递红石信号 |
| 能量拴绳 | 铁块 | 传输 FE 能量 |
| 物品拴绳 | 漏斗 | 在容器间运输物品 |
| 流体拴绳 | 炼药锅 | 在流体容器间运输流体 |
| 加压管道拴绳 | Mekanism 钢块 | 运输 Mekanism 化学品/气体 |
| 热导管道拴绳 | 铜块 | 在 Mekanism 设备间均衡热量 |
| ME 线缆拴绳 | AE2 福鲁伊克斯水晶块 | 桥接 AE 网络 |

也可以在物品栏中转换：一只手拿超级拴绳，另一只手拿转换材料并使用，会转换其中 1 根绳。

### 5. 设置运输方向

物品、流体和加压管道必须指定抽取端；能量绳也可以指定抽取端以改为定向输送。

转换完成后，再用该类型的**转换材料**潜行右键绳索，并把准星指向靠近来源的一端，即可切换该端的抽取状态：

- 能量：铁块
- 物品：漏斗
- 流体：炼药锅
- 加压管道：Mekanism 钢块

再次操作同一端可取消抽取方向。热导管道与 ME 线缆不需要设置方向。

## 操作速查

| 想做什么 | 操作 |
| --- | --- |
| 架设绳索 | 手持超级拴绳，依次右键两个方块面 |
| 放置时延长 | 选好第一锚点后，再次右键同一锚点 |
| 转换、升级或切换抽取端 | 潜行 + 对准绳体 + 对应材料右键 |
| 剪断单根绳 | 潜行 + 剪刀右键绳体 |
| 剪断栅栏结上的绳 | 剪刀右键栅栏结 |
| 添加挂件 | 主手拿挂件、副手拿线，非潜行右键绳体 |
| 切换方块挂件形态 | 左键挂件 |
| 取下单个挂件 | 潜行 + 剪刀左键挂件 |
| 调整挂件显示 | 挂件调节器右键挂件 |
| 编辑告示牌挂件 | 空手右键告示牌挂件 |
| 使用滑索 | 手持锁链右键已启用滑索的普通/红石绳 |
| 脱离滑索 | 按 Shift |
| 打开本地配置 | 在控制设置中绑定快捷键，或输入 `/superlead gui` |

> **为什么强调潜行？** 非潜行右键绳索优先用于添加挂件；转换、升级、剪断和方向操作需要潜行，以免一箱材料突然变成晾衣绳装饰。

## 绳索类型详解

### 普通拴绳

普通拴绳适合装饰、悬挂物品和物理玩法。启用对应物理预设后，它还可以支持滑索、玩家接触与鹦鹉停留。

普通绳本身不传输资源。

### 红石拴绳

红石拴绳会在同一连通网络中收集外部红石信号，并把**最高信号强度**传播到网络中的所有端点，范围为 0–15。

适合：

- 跨过不方便布线的空间；
- 在多个端点间同步信号；
- 制作可视化机关和通电滑索。

红石绳没有等级。其信号是双向传播的，不需要设置输入端。

### 能量拴绳

能量拴绳传输 NeoForge FE 能量。

单根绳的默认吞吐为：

$$
256 \times 2^{等级}\ \text{FE/t}
$$

使用**红石块**升级等级。升级材料需求随当前等级增加：0 → 1 需要 1 个，1 → 2 需要 2 个，2 → 3 需要 4 个，以此类推。

能量网络有两种模式：

- **自动均衡**：网络中没有指定抽取端时，能量会从较满的设备流向较空的设备。
- **定向输送**：只要网络中有任意能量绳指定了抽取端，整个连通网络就只从标记的来源向其他端点供能。

绳上出现发光或脉冲，表示最近发生过能量传输。

为防止某些科技模组的能量接口在极高等级下拖慢或卡死服务器，能量网络带有服务端安全保护：单次 FE 请求、每网络尝试次数、每维度调用次数和处理时间都有预算。接口抛出异常、返回非法数据或响应过慢时，对应端点会暂时熔断并在冷却后自动重试。高等级仍会提高网络吞吐额度，但系统不会为了在一个 tick 内耗尽额度而无限调用机器。

### 物品拴绳

物品拴绳必须使用漏斗设置抽取端。默认每 4 tick 尝试一次传输，单次批量为：

$$
\min(64, 2^{等级})\ \text{个物品}
$$

使用**箱子**升级等级。默认最高等级为 6，即最高 64 个物品/次。

多根物品绳通过共同端点组成网络后，会沿同类型绳索寻找目标，并在多个分支间轮询。运输路径最多经过 64 根绳。

需要注意：目标必须能接收本次抽出的完整批量，否则该次传输不会提交。高等级绳遇到空间不足的小容器时，可能因此暂停运输。

### 流体拴绳

流体拴绳必须使用炼药锅设置抽取端。默认单次传输量为：

$$
1000 \times 2^{等级}\ \text{mB}
$$

使用**空桶**升级等级。默认最高等级为 4，即 16000 mB/次。

它支持 NeoForge 流体能力；安装 Mekanism 后，也会尝试连接 Mekanism 的流体设备。

### 加压管道拴绳（Mekanism）

需要安装 Mekanism。它同时支持 Mekanism 统一化学品 API 中的气体和其他化学品。

- 转换：钢块；
- 设置抽取端：再次使用钢块；
- 升级：强化合金；
- 默认基础批量：1000，每级翻倍；
- 默认最高等级：4。

网络会轮询多个目标，也可以通过挂件样本进行无界面过滤。

### 热导管道拴绳（Mekanism）

需要安装 Mekanism。热导管道不是单向运输，而是让同一网络内的热能力端点趋向均衡，因此不需要设置抽取端。

- 转换：铜块；
- 升级：强化合金；
- 默认基础速率：每根 1000 热量单位/t，每级翻倍；
- 默认最高等级：4。

连通组件的总传热上限为其中所有热导绳速率之和。

### ME 线缆拴绳（AE2）

需要安装 Applied Energistics 2。ME 线缆拴绳会使用 AE2 官方 Grid API 连接两端网络节点，并遵守 AE2 自身的供能与频道规则。

- 转换：福鲁伊克斯水晶块；
- 默认容量：8 频道；
- 升级：16³ 空间组件；
- 升级后容量：32 频道。

它不会提供 256 频道，也不会绕过 AE2 的频道限制。

可以把普通 **ME 终端**（`ae2:terminal`）挂到 ME 线缆拴绳上，空手右键即可打开 AE2 原生终端界面。当前仅普通 ME 终端可作为功能终端使用；合成终端、样板终端等暂不开放对应菜单。

## 网络、分流与过滤

### 栅栏结和分支

多根同类型传输绳可以在共享端点处构成网络。物品、流体和加压管道网络会在可用分支间轮询，适合制作轻量分流系统。

不同类型的绳不会混合传输。例如，物品路径中间接一根普通绳，物流不会跨过去。

### 用普通挂件过滤

挂件不只是装饰。在传输绳上挂载有效资源样本，可把它变成过滤器：

- 物品绳：任意普通物品都可作为物品样本；
- 流体绳：只有能读取出流体的容器才参与过滤，例如水桶；
- 加压管道：只有能读取出 Mekanism 化学品的容器才参与过滤。

一根绳上有多个有效挂件时，满足其中任意一个即可通过；整条路径上每一根带过滤条件的绳都必须允许该资源。

因此可以在不同分支上挂不同样本，实现完全不打开 GUI 的分类路由。

### 基础货单

合成方式：**纸 + 线**，无序合成。

右键打开后，可以配置：

- 18 个物品样本槽；
- 白名单或黑名单；
- 是否精确匹配物品组件（界面显示为 NBT 开关）。

样本槽是幽灵槽，不会真正吞掉物品。将配置好的货单挂到物品绳上即可生效。

### 高级货单

合成方式：**基础货单 + 书**，无序合成。

除基础功能外，还可加入最多 32 个物品标签，例如 `minecraft:logs`。安装 JEI 后，可以直接把 JEI 中的配料拖入样本槽。

## 挂件与装饰

### 添加挂件

1. 主手拿要悬挂的物品；
2. 副手拿线；
3. 不要潜行，瞄准绳索右键。

成功后会消耗挂件物品和线各 1 个。大多数物品都可以挂载，包括灯笼、告示牌、方块和工具等。

### 调整和拆除

- 左键方块挂件：在方块显示与物品显示之间切换；
- 潜行 + 剪刀左键挂件：只取下该挂件；
- 挂件调节器右键：调整上下位置、缩放、朝向、吊绳、主绳穿过方式和方块状态；
- 剪断整根绳：其上的挂件一并掉落。

挂件调节器使用 **铁锭 + 红石粉 + 线** 无序合成。

### 告示牌挂件

告示牌挂到绳上后仍可交互：

- 空手右键：编辑文字；
- 染料右键：改变文字颜色；
- 荧光墨囊右键：切换发光文字。

### 装饰物也可能影响运输

物品绳上的普通物品挂件会被当成过滤样本。如果你只想装饰，不想限制物流，请把装饰挂在普通绳上，或确认它符合你的路由意图。

## 剪断、返还与拆迁

- 潜行并用剪刀右键绳体，可剪断单根绳；
- 用剪刀右键栅栏结，可剪断结上的所有绳；
- 生存模式会消耗剪刀耐久；
- 绳上的挂件会一并掉落；
- 升级绳会按服务器的返还比例掉落部分升级补偿材料，默认比例为 50%。

> 当前 beta 版本的主动剪断返还存在已知异常：有时可能同时掉落普通超级拴绳和保留类型的超级拴绳。请以实际掉落为准，服务器管理者也应留意这一行为。

## 物理预设、滑索与鹦鹉

这些玩法需要绳索绑定一个启用了相应选项的物理预设。默认新放置的绳并不会自动成为滑索。

### 创建玩家预设

预设绑定器的合成方式为：**纸 + 线 + 红石粉 + 铁锭**，无序合成。

1. 手持未绑定的预设绑定器右键，创建一个玩家预设；
2. 再次右键可编辑参数；
3. 开启“启用物理”，并按需开启“玩家滑索”“鹦鹉寻路”“玩家反推”或“玩家绊倒”；
4. 手持该绑定器，潜行右键绳索，将绳加入或移出预设管理。

服务器可以关闭预设功能。管理员也可以创建物理区域，让区域内的新绳自动绑定指定预设。

### 滑索

滑索只适用于绑定了滑索预设的**普通绳或红石绳**。

1. 手持铁链或受支持的铜链；
2. 靠近绳索并右键；
3. 骑行时继续在任一只手中持有锁链；
4. 按 Shift 脱离。

滑索会沿下坡加速。通电红石绳拥有额外加速能力，并可在共享结点处续接到合适的下一根滑索。

### 鹦鹉停留

开启预设中的“鹦鹉寻路”后，附近处于合适状态的鹦鹉可能飞到绳上停留。

潜行并手持小麦种子、南瓜种子、西瓜种子、甜菜种子、火把花种子或瓶子草荚右键绳索，可暂时提高鹦鹉选择这根绳的概率。玩家快速贴近时，鹦鹉会受惊飞走。

### 玩家接触和绊倒

预设还可启用：

- 玩家反推：绳索像柔软障碍一样推开接触者；
- 玩家绊倒：玩家越过绳索时有概率进入爬行并受到 1 点伤害。

这些选项默认关闭，主要用于机关、冒险地图或整活场景。

## 客户端配置与性能

在控制设置中为“打开 Super Lead 调试面板”分配按键，或输入 `/superlead gui` 打开配置界面。默认快捷键未绑定。

可调整的内容包括：

- 绳索松弛、重力、阻尼和碰撞；
- 绳间排斥、风力和阵风；
- 各类型绳索颜色、粗细与 3D/彩带渲染；
- 物理激活距离、最大渲染距离与 LOD；
- 动态物理预算和静止步进间隔；
- 挂件缩放与下垂。

本地配置保存在 `config/super_lead-tuning.properties`，只影响当前客户端的视觉和本地物理表现。服务器预设可覆盖绑定绳索的相关参数。

如果帧率下降，优先尝试：

1. 降低“动态物理预算”；
2. 缩短“物理激活距离”；
3. 提高静止绳的步进间隔；
4. 提前切换为彩带渲染；
5. 缩短最大渲染距离。

## 管理员简明指南

### 服务端配置

主配置文件为 `config/super_lead-common.toml`。常用项目包括：

| 配置 | 默认值 | 作用 |
| --- | ---: | --- |
| `energy.base_transfer_per_tick` | 256 | 0 级能量绳 FE/t |
| `energy.tier_max_level` | 30 | 能量绳最高等级 |
| `energy.max_request_per_call` | 65536 | 单次传给第三方接口的最大 FE |
| `energy.max_attempts_per_component_tick` | 32 | 单个能量网络每 tick 最大传输尝试数 |
| `energy.max_handler_calls_per_level_tick` | 2048 | 每维度每 tick 的 FE 接口调用预算 |
| `energy.tick_budget_micros` | 2000 | 每维度 FE 处理软时间预算（微秒） |
| `energy.slow_call_threshold_micros` | 500 | 超过该耗时的端点将触发熔断 |
| `energy.breaker_cooldown_ticks` | 200 | 故障端点的默认冷却时间 |
| `network.max_leash_distance` | 12.0 | 每个长度单位的最大格数 |
| `network.max_ropes_per_block_face` | 8 | 单个方块面最多连接数 |
| `network.item_transfer_interval_ticks` | 4 | 物品、流体和化学品的传输间隔 |
| `network.fluid_bucket_amount` | 1000 | 0 级流体绳单次 mB |
| `network.cut_refund_ratio` | 0.5 | 剪断后的升级材料返还比例 |
| `presets.allow_op_visual_presets` | true | 是否启用服务器预设系统 |

在本地配置界面点击“服务器配置”，可管理服务器项、预设包和物理区域。默认查看需要 OP 2，修改高风险服务器参数需要 OP 4 或对应权限节点。

### 创建物理区域

1. 打开服务器配置界面的“物理区域”页，点击“剪刀框选”；
2. 使用剪刀潜行右键两个方块，选择区域对角；
3. 输入区域名并选择已有预设；
4. 创建后，区域内新放置或与区域相交的绳会绑定该预设。

也可使用 `/superlead zone select` 开始框选。

| `/superlead serverconfig list` | 列出服务器配置 |

## 常见问题

### 绳连上了，为什么不传东西？

依次检查：

1. 两端是否是正确的容器或机器面；
2. 绳索类型是否正确；
3. 物品、流体或加压管道是否设置了抽取端；
4. 路径中是否混入其他类型绳索；
5. 挂件或货单是否过滤了资源；
6. 目标是否有空间完整接收本次批量；
7. 路径是否超过 64 根绳；
8. 相关区块是否加载。

### 能量为什么自己反向流动？

没有指定抽取端时，能量网络处于自动均衡模式。若要固定方向，请用铁块潜行右键靠近来源端的能量绳。

### 为什么材料被挂到了绳上，没有升级？

转换、升级和方向切换都需要潜行。非潜行操作会优先尝试添加挂件。

### 为什么锁链右键不能滑？

确认绳索为普通绳或红石绳，并且已绑定启用了“启用物理”和“玩家滑索”的预设。玩家还必须靠近绳索，并在骑行期间继续持有锁链。

### 为什么快捷键没有反应？

默认快捷键未绑定。请到控制设置中手动分配，或直接使用 `/superlead gui`。

### 挂件调节器怎么制作？

使用 **铁锭 + 红石粉 + 线** 无序合成，即可得到 1 个挂件调节器。

### 绳为什么突然消失？

服务端会定期清理无效连接。端点方块被破坏、连接超出允许距离，或普通方块挂载面被完整方块堵住，都可能使连接失效。

## 最后的小建议

- 先用两只箱子练习一根物品绳，再搭复杂网络；
- 在分支处挂样本，比猜物流方向直观得多；
- 装饰绳和物流绳尽量分开，避免挂件意外成为过滤器；
- 大型基地优先规划方块能力面与区块加载范围；
- 想搭滑索公园，先做好预设，再批量绑定绳索。

祝你玩得开心。毕竟管道会堵，线缆会乱，但绳子至少乱得很好看。
